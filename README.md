# Prueba Técnica Generador Excel

--- 

### Pasos para la instalación del proyecto

 > Crear la base de datos en su entorno local, tomar la base de datos **credits_subscription.sql** proporcionada en el proyecto.

 > Ingresar las claves correctas en el archivo .env, así como crear su **APP_KEY** con el comando `php artisan key:generate` 

 > Instalar las dependencias del proyecto con los comandos 
    `composer install` y `npm install`


### Montaje del proyecto

 > Correr el work de laravel con el comando `php artisan qeue:work`

 > Correr el proyecto con el comando `composer run dev`

#### Listo! el proyecto podrá descargar el excel
